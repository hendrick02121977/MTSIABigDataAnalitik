## Course Learning Outcomes (CLO)
1. Mahasiswa mampu menjelaskan konsep Big Data dan penerapannya dalam akuntansi digital.  
2. Mahasiswa mampu menggunakan Google Colab dan Python untuk analisis data akuntansi.  
3. Mahasiswa mampu mengimpor, membersihkan, dan mengolah dataset akuntansi.  
4. Mahasiswa mampu membuat visualisasi dan dashboard akuntansi interaktif.  
5. Mahasiswa mampu menerapkan machine learning untuk prediksi dan fraud detection.  
6. Mahasiswa mampu mengintegrasikan Big Data tools (Spark, BigQuery) dalam analisis akuntansi.  
7. Mahasiswa mampu menyajikan hasil analisis dalam bentuk laporan dan presentasi profesional.
